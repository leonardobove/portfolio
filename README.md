# Leonardo Bove — Personal Portfolio

A modern, responsive personal portfolio built with the vCard template, showcasing quantum computing and electronic engineering research and projects.

## About

Graduate Research Fellow at the University of Pisa, specializing in:
- **Superconducting Qubits** — Characterization and optimization for quantum computing
- **FPGA & Quantum Control** — Real-time control systems and quantum state manipulation
- **Electronic Instrumentation** — Cryogenic control and measurement electronics
- **Quantum Research** — Academic research and publications in quantum computing

## Features

- **Responsive Design** — Mobile-first, works seamlessly on all screen sizes
- **Dark/Light Theme Toggle** — User preference for theme selection
- **Smooth Navigation** — Single-page layout with smooth scrolling
- **Portfolio Showcase** — Project gallery and publications
- **Contact Section** — Easy reach via email and social links

## Getting Started

### Local Development

```bash
# Clone the repository
git clone https://github.com/leonardobove/portfolio.git
cd portfolio

# Start a local server (Python 3)
python3 -m http.server 8000

# Open in browser
# http://localhost:8000
```

### Deployment

The site is built with GitHub Pages. Push to `main` branch and GitHub Actions will automatically build and deploy.

```bash
git add .
git commit -m "Your message"
git push origin main
```

Live site: https://leonardobove.github.io/

## Customization

### Profile Information
Edit the following in `index.html`:
- Profile name, title, location (sidebar)
- Contact email and social links
- About section text
- Skills and proficiency levels
- Projects and portfolio items

### Styling
- CSS: `assets/css/style.css`
- Modify colors, fonts, spacing as needed

### Assets
- Profile avatar: `assets/images/my-avatar.png`
- Project images: `assets/images/project-*.png`
